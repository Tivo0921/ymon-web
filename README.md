\# YNU Monsters (MVP)



\## Overview

Simple 3v3 battle game with matchmaking.



\## Stack

\- Next.js (App Router)

\- Express

\- Supabase (Postgres)



\## Current Features

\- User create/get by handle

\- Matchmaking (idempotent, race-safe)

\- 3v3 battle simulation

\- Battle result stored in DB



\## Local Setup



\### API

cd apps/api

npm install

npm run dev



\### Web

cd apps/web

npm install

npm run dev
